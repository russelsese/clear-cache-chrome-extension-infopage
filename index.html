Everything you need to manage browser data

14 data types, 5 time ranges, custom presets, scheduling, and domain rules — all free.

🗑️

One-Click Clearing

Hit the toolbar icon and your data is gone — no confirmation dialogs, no settings navigation. Just clear.

📋

Presets

Save named clearing configurations. Built-in presets for Cache Only, Developer, Privacy, and Full Clean. Create unlimited custom presets — free, always.

⏰

Automatic Scheduling

Set a recurring interval (every N minutes) or a daily time. Enable clear-on-startup to start every session fresh. All completely free.

🎯

Active Tab Only

Clear just the data for the site you're currently on. Perfect for developers who need to test one origin without wiping everything.

🔒

Domain Allowlist & Blocklist

Protect your banking logins while clearing everything else. Set per-type rules — different allowlists for cookies versus cache.

👁️

Preview Before Clearing

See exactly what will be deleted — data types, time range, and scope — before you confirm. No surprises.
