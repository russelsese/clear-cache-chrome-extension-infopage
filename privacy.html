Privacy Policy

Last updated: April 2026

The short version

Clear Cache Free collects no personal data, sends no data to any server, and includes no analytics or tracking of any kind. Your browsing data is accessed only to delete it — it is never read, stored externally, or transmitted.

What data the extension accesses

Clear Cache Free requests the following Chrome permissions. Each is used exclusively for the function described — nothing else.

browsingData

Required to clear cache, cookies, history, and other browsing data. The extension calls the chrome.browsingData API to delete data you select. It does not read or inspect the contents of that data before deleting it.

storage

Your settings, custom presets, domain rules, and schedule configuration are saved locally using chrome.storage.sync. This data syncs across your own Chrome profiles via Google Sync if you are signed in — it is never sent to any third-party server.

alarms

Used exclusively to trigger scheduled clears. No data is transmitted when an alarm fires.

notifications

Used to display an optional confirmation notification after a clear completes. The notification contains only the list of data types that were cleared. This information is shown only on your device and is not recorded.

cookies

Required by the chrome.browsingData API to perform domain-aware cookie clearing. The extension does not read individual cookie values.

host_permissions (<all_urls>)

Required to support Active Tab Only mode and domain-specific clearing rules. The extension does not read page content — this permission is needed solely to identify the origin of the active tab when scoping a clear operation.

Data collection & analytics

Clear Cache Free contains no analytics SDK, tracking pixel, telemetry endpoint, or crash reporter. The extension makes zero outbound network requests. You can verify this by inspecting the source code.

  • No usage statistics are collected
  • No install or uninstall events are reported
  • No feature usage is logged
  • No device identifiers are recorded
  • No IP addresses are stored

Data storage

The only data Clear Cache Free stores is your preferences (settings, presets, schedules, domain rules). This is saved using chrome.storage.sync, which is managed entirely by Chrome and Google's sync infrastructure. Clear Cache Free has no access to the storage layer beyond reading and writing your own preferences.

You can delete all stored data at any time by uninstalling the extension or clearing the extension's storage via Chrome's developer tools.

Third-party services

Clear Cache Free does not integrate with, embed, or communicate with any third-party service. There are no:

  • Advertising networks
  • Analytics platforms (Google Analytics, Mixpanel, Amplitude, etc.)
  • Error monitoring services (Sentry, Bugsnag, etc.)
  • Remote configuration services
  • Content delivery networks serving extension code

All extension code is bundled and shipped with the extension package. No code is loaded from external URLs at runtime.

Children's privacy

Clear Cache Free does not knowingly collect any information from anyone, including children under 13. Because no data is collected at all, there is no special handling required.

Changes to this policy

If this policy changes, the updated version will be posted on this page with a new "Last updated" date. Given that the extension collects no data, material changes to this policy are unlikely.
